# Security Policy

## Supported Versions

Currently, only the latest version of AugmentedQuill is supported for security updates.

## Reporting a Vulnerability

If you discover a potential security vulnerability in AugmentedQuill, please do not open a public issue. Instead, please report it privately to the maintainers.

We will acknowledge receipt of your vulnerability report within 48 hours and strive to provide a fix or mitigation plan as soon as possible.
